# 一场差点取消的活动，是如何变成我们所有人的高光时刻的？

我本以为，这只是又一个成年人世界里，关于“散场”的寻常故事。

我们团队每两个月的活动，这次轮到我和小H做PM。临近日期，群里却静悄悄。各种理由汇集而来，最后，报名表上孤零零地躺着五个名字。

说实话，我内心毫无波澜。作为一个“老年人”，我太理解这种状态了。生活、工作、家庭……每个人都有自己的轨道和引力，曾经的交集越来越少，这是自然规律。我甚至已经准备好了安慰自己的说辞：天下没有不散的宴席，情谊在心里就好。

我接受了这个剧本，直到小H的电话打了进来。

电话那头，他的声音带着一股不甘心。他很直接，说想取消这次活动，因为他不满意，他觉得我们可以做得更好，应该有更多人来。起初，我还在用那种过来人的、理性的、甚至有点冷淡的语气跟他分析。我条理清晰，逻辑分明，可能在辩论上，他完全说不过我。

但他没有跟我辩论。他只是非常、非常真诚地告诉我，他有多失望，他有多不希望看到团队就这么慢慢冷下去。他没有指责任何人，他只是把他内心所有的渴望，毫无保留地摊开在我面前。

那一刻，我所有的“道理”都显得苍白无力。他反复表达的，其实只有一个意思，一个直击灵魂的问题：**“凭什么？？？”**

凭什么我们就该接受这个“越来越冷”的结局？我们为什么不能做得更好呢？他告诉我，他不想认命。就这四个字，像一道闪电，劈开了我自以为是的“佛系”外壳。我被他那种不服输的、纯粹的生命力点燃了。我脱口而出：“你有没有想过把这些话告诉大家？你说的这么真诚，一定能打动他们！”

挂掉电话，我几乎是立刻就想到了小W。她是团队里的KOL，是那个能撬动全局的关键人物。

**深夜十一点，拨通她的电话时，我的心其实是悬着的。果不其然，当我把情况和想法有点语无伦次地告诉她后，她还是拒绝了。理由很充分，行程早已安排好，无法变动。**

**就在我快要放弃，准备说“好的打扰了”的时候，被小H点燃的那团火让我多坚持了一下。我没有再强求，而是把小H那种“不认命”的真诚和我们内心的失落，原原本本地传递了过去。我告诉她，我们不是在道德绑架，只是真的、真的很希望能和大家再热闹一次。**

**电话那头沉默了。但这次的沉默和之前的不同，我能感觉到，她不是在想怎么拒绝，而是在思考。然后，奇迹发生了。我能清晰地感觉到，她在被我们的情绪感染后，开始主动地把这件事当成自己的事来想。她开始问一些细节，帮我们分析问题出在哪里。她的角色，从一个“被邀请者”，悄然转变成了一个“局内人”。**

**就在这来来回回的拉扯和共情中，她突然说：“等等，我有个想法……既然原来的方案不行，我们干脆推倒重来。要不……你们来我家吧？我们通宵聊！”**

我愣住了。这就是统合综效吗？最终的完美方案，不是我的提议，也不是小H的愿望，而是在我们用真诚敲开她心门后，由她亲手创造出的、一个让所有人都意想不到的、更棒的可能性。她不仅加入了，还把自己的家变成了我们新的据点。

那个深夜，我们三个人，对着屏幕，你一言我-语，兴奋地把整个行程推倒重来。窗外夜色渐深，但我们心里的火却越烧越旺。我们知道，我们正在创造一个奇迹。

后来的故事，顺理成章。新的方案发出去，小W的加入像一块磁铁，把原本犹豫不决的小伙伴们一个个吸了回来。我们的队伍不断扩大，活动的内容也变得比原计划精彩百倍。

我们不仅挽救了一场活动，更重要的是，我们重新点燃了彼此之间的那份链接。这次小小的蝴蝶效应，让我看到了一个看似“无解”的局面，是如何被一个人的“不认命”所撬动，被一群人的“不放弃”所改变的。

故事讲完了，但真正改变我的，是故事背后的一些东西。它颠覆了我过去长久以来信奉的一个人生准则。

关于那次颠覆性的认知跃迁，我们下篇再聊。
