# 一次做Agent大赛评委的感受

今年在年底参与了一件大事就是星瀚律所举办了全国法学生的第一节Agent大赛，吸引了全国各大高校（虽然主要是文科类强的高校）的学生来报名参加，一共有好像60组队伍吧，新华网也报道了非常厉害。我在其中有幸做了技术评委，也算贡献了一点点自己的力量。

作完之后几点很深刻的感受这里一定要记录一下。首先就是同学们不要再卷啦，睁开眼睛看看世界吧。现在大家都愁找工作，但是决赛的6只队伍在路演后N多大企业都是直接发offer的！抢着要啊 。你还在那里刷题学一些过时的东西，千军万马挤独木桥，是不是太low了。需求多的很，只要你有一双发现的眼睛。从我的角度来看这些决赛的作品虽然出色但是作为大学生我觉得还是可以做的更好的 并不是遥不可及，特别他们如果能找到我这样的人指导一下，妥妥的脱颖而出。所以真的是怎么利用好资源才是最重要的！你要让企业能看到你，还有什么比这种法律界的第一次agent大赛更好的机会呢？技术界的不好拿，大神太多，找个细分赛道你就是麻雀变凤凰啊。呵呵，有点像在新疆去参加高考，高考游民。总之就是用智商来反卷！

在一点就是我自己的心态。我会以我的眼光看大学生的作品，觉得不够出色，认为他们可以做到更好。但是忽略了他们的真实环境和水平，忽略了我的世界知识。所以以后做评委要平常心一些，能够站在历史当时的视角去看待当时的决策才是好的评价不是吗？中国的大学生在现在的教育环境中能做到这样的创新已经很不错了！

最后一点收获就是第一次从内部去看一个大型比赛的筹备过程，而且这里面有一些优秀的人，让我也提高了审美，知道了什么是最好的。表面上看比赛最后在交大的法学院举行，大佬云集好不风光。外面的人肯定会想这要投入很多钱给了好多资源吧，我肯定没有资源我做不来。但是我从内部看我参与其中我才知道，任何一件事是你能在有限的资源下把它做的超出预期那才叫真的好，才有可能获得更多的资源。**很多时候第一次并不是你有了资源才做的好，而是你做的好才有了资源。**听了我们比赛主理人宋老师的复盘我才略知一二。你以为你想举办一个比赛大家就会来报名，但是实际上可能因为早期没有知名度很少人报名，那怎么办？谁来想办法，你呀！你要发挥一切的人脉让更多的人帮你，不要指望别人。然后大家报名了，但是法学生agent这种都没听过，完全不会技术怎么办？最后的决赛开天窗吗还是低水平的角逐？怎么办？你要想办法呀，你要给这些报名的大学生做培训帮他们拉资源组队教会他们，也不要指望别人！宋老师说当时不知道大学生现在都喜欢最后一天交作业。所以到了提交截止日之前只有6份，急都急死了，那个愁的人是谁，也是你呀！还好最后一天作品像雪片一样的飞来，比赛可以如期举行了，心里的石头才放下来。这就是第一次的痛苦也是第一次的魅力。不得不佩服优秀的人就是优秀。很少有一个团队我能在内部看比在外部看更加佩服的。大家都懂得普遍状态是金玉其外败絮其中，呵呵呵。所以说，好好体验人生吧，做万件事才是真正的学习。
